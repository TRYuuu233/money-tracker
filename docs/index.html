Features

Everything You Need. Nothing You Don't.

No bloat. No subscription. Just beautifully crafted finance management.

📚
Unlimited Ledgers
Create separate ledger books for different purposes — personal, business, travel — each with its own icon, color, and balance.
📊
Real-time Analytics
Animated net worth hero card. Add a transaction and watch the numbers update instantly — no refresh needed.
🔍
Smart Sort & Filter
Filter by All / Income / Expense. Sort by date or amount, ascending or descending. Works instantly on thousands of records.
🌐
Bilingual ZH / EN
Switch between Chinese and English in Settings. Full UI adaptation — every label, button, and dialog.
🔐
Password Protection
Optional startup password lock. Your finances stay private even if others use your computer.
📤
Excel Export
Export any ledger to a beautifully formatted .xlsx file. Perfect for monthly reviews or sharing with an accountant.
🔎
Global Search
Instantly search transactions across ALL your ledgers simultaneously. Find any record in milliseconds.
🎨
Premium Dark UI
Glassmorphism design, smooth hover animations, glow effects, and animated number transitions. Finance has never looked this good.

Get Started

Up and Running in 60 Seconds

No installer. No account. Just Python and two packages.

1. Install dependencies

$ pip install PyQt6 openpyxl

2. Clone and run

$ git clone https://github.com/TRYuuu233/money-tracker
$ cd money-tracker
$ python main.py

Or download the pre-built .exe from Releases
