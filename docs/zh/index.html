📚
多账本管理
无限创建独立账本，各自拥有图标、颜色和余额，个人、工作、旅行分开管理。
📊
实时动画分析
净资产大卡片 + 各账本明细拆分，记账后数字立刻动态更新，无需手动刷新。
🔍
排序筛选
按全部/收入/支出筛选，按日期或金额升降序排列，海量记录一秒定位。
🌐
中英双语界面
系统设置中切换中文/英文，所有标签、按钮、弹窗完全适配两种语言。
🔐
启动密码保护
可选设置启动密码，他人无法查看你的财务数据，隐私更有保障。
📤
导出 Excel
任意账本一键导出为格式精美的 .xlsx 文件，方便备份或与他人分享。
🔎
全局搜索
跨所有账本同步搜索交易记录，毫秒级响应，快速找到任意一笔账。
⌨️
快捷键记账
自定义快捷键，直接在任意界面快速呼出收入/支出记录窗口，效率倍增 。

快速开始

60 秒内启动运行

无需安装包，无需注册账号，只需 Python 和两个依赖包。

① 安装依赖

$ pip install PyQt6 openpyxl

② 克隆并运行

$ git clone https://github.com/TRYuuu233/money-tracker
$ cd money-tracker
$ python main.py

也可直接从 Releases 页面 下载编译好的 .exe
